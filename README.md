# Simple Web Scraper (Python)

A basic Python web scraper that fetches a webpage and extracts:

- Page title
- Page body text
- All links (href attributes)

It uses the `requests` library to download the webpage and `BeautifulSoup` to parse HTML content.

---

## 🚀 Features

- Fetch webpage content from a URL
- Extract and display:
  - Page title
  - Main text content
  - All hyperlinks
- Clean and simple command-line usage
